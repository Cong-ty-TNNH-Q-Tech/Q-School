### 🚨 Bắt buộc phải sửa (Request Changes):

1. **Sai cấu trúc Response chuẩn (API Envelope)**
   - Theo quy định trong `AGENTS.md` và `openapi.yaml`, mọi API cần trả về định dạng bọc ngoài: `{"status": "success", "data": {...}}`. Dự án đang dùng `ApiResponse` (xem `classes.py` làm chuẩn).
   - Hiện tại trong `quizzes.py` và `essays.py`, các API đang return thẳng data model (VD: `return EssaySubmissionAcceptedResponse(...)` với `response_model=EssaySubmissionAcceptedResponse`). 
   - **Cách fix**: Bạn cần bọc tất cả endpoint lại với `ApiResponse`. Ví dụ: `response_model=ApiResponse[QuizAttemptResponse]` và `return ApiResponse(data=await use_case.start_attempt(...))`.

2. **Xử lý Exception trong `essays.py` chưa đúng chuẩn**
   - Trong `get_essay_submission`, bạn đang ném trực tiếp `AppException` với `status_code` hardcode:
     ```python
     from app.domain.exceptions import AppException
     raise AppException("B├ái nß╗Öp kh├┤ng tß╗ôn tß║íi", status_code=404)
     ```
   - Việc này vi phạm cách xử lý chuẩn. Ở tầng Router, bạn nên import và sử dụng `NotFoundException`, `ForbiddenException` từ `app.core.exceptions` (như bạn đã làm rất chuẩn chỉnh ở bên file `quizzes.py`).
